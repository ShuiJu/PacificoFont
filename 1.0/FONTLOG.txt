Basic Font Information

‘Pacifico’ is a design derived from advertisement lettering from the 1940's. The lettering was used to inspire and develop Pacifico, which now supports most European Latin languages and OpenType Ligatures and extra features.

This release supports the following 
Unicode ranges: Latin, Latin-Extended

There are 3 Source files:

1. src/Pacifico.sfd Original Source Files in postscript curves.
2. src/Pacifico.ufo Source Files in UFO format.
3. src/Pacifico.otf OpenType font format.

To contribute to the project contact Vernon Adams
at vern@newtypography.co.uk and use the gitHub repository at https://github.com/vernnobile/PacificoFont
